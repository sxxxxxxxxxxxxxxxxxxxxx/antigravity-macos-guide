怎么对号（先看这三句）

• 账号没资格：先查年龄认证、地区与订阅资格，不要先反复注入。

• oauth2 连接失败 / 网络超时：就是客户端没真正走通代理——请按 第三章网络配置 处理（Tun / Antify 等）。

• Further action / 扫码：登录阶段人机验证，走扫码流程；年龄认证在「账号没资格」那条处理。

• 对话时报 400 且含 User location is not supported：出口地区不被支持，先换日/新等干净节点再重开客户端（详见 FAQ 异常 3）。

• 只要 VS Code 界面：下 Antigravity IDE；发消息一直转圈：优先当网络未接管。
